# Hugo Website

## Install hugo
```
winget install Hugo.Hugo.Extended
```
## Clone my hugo repo
```
git clone https://github.com/SilentSmeary/hugo.git
```
## Run the server
```
cd hugo
```
```
hugo server
```

## Open the website
Head to http://localhost:1313/ and browse the website
